Контрольный расчет для `n` точек

	Назначение:
		Проводится однократный расчёт значений входного (Uvx)
		и выходного (Uvix) сигналов в n равномерно
		распределённых точках на заданном интервале времени.

	Роль переменной n:
		Определяет, в скольких точках будет вычислен сигнал.
		Значение n вводит пользователь вручную.
		Это позволяет проверить корректность расчётов,
		оценить форму сигнала и сравнить
		с ожидаемыми результатами.

	Характер расчета:
	
		Одноразовый.

		Точность задана числом n,
		без автоматического уточнения.

		Используется для визуализации
		и первичной отладки.
