Контрольный расчет для n точек

Назначение:
	Однократное вычисление входного (Uvx) и выходного (Uvix) сигналов в n точках на заданном временном интервале.

Параметр n:
	Задаётся пользователем. Определяет количество точек расчёта для проверки корректности и визуализации сигнала.

Особенности:

	Одноразовый расчёт без автоматического уточнения

	Точность зависит от n

	Используется для отладки и анализа формы сигнала
