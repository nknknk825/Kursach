Краткое пояснение к параметрам n и eps
--------------------------------------

Расчёт с фиксированным n:
- n — число разбиений отрезка [a, b].
- Используется для разового расчёта интеграла с шагом h = (b - a) / n.

Расчёт с точностью eps:
- eps — допустимая погрешность.
- n — стартовое число разбиений.
- Метод уточняет результат, пока разность между расчётами с n и 2n меньше eps.
Условие остановки: |I_2n - I_n| < eps
