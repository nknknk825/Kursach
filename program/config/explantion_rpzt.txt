Автоматический расчёт с точностью `eps`  

Суть:  
	Итерационный поиск значения параметра (например, длительности импульса) с контролем погрешности.  

Ключевые переменные:  
	- `eps` — допустимая погрешность (например, 0.01 = 1%)  
	- `n` — стартует с малого (например, 11), удваивается до достижения точности `eps`  

Особенности:  
	- Многошаговый процесс с автоматическим уточнением  
	- Цель — гарантировать точность результата  
	- Применяется для финальных расчётов
