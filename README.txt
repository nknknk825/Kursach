Чтоб скомпилировать си программу надо в папки program вести: make. Для первого запуска тоже требуется её прописать.
Программа запускается из папки program командой: make run
Сооветую ознакомится с кодом, т.к Хазиев может начать задавать вопросы к содержанию программы. Будут вопросы или предложения пиши)
